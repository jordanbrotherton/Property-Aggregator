# Property Aggregator
![Screenshot of Property Aggregator](resources/images/Screenshot.png)
Look through Alachua County property tax records!

## Features
Property Aggregator provides these features for browsing properties:
* Filter out properties through minimum price/size.
* Show/hide properties with unassigned addresses.
* Open a Google Maps search of the address by clicking on the card.
* Use a B+ or Heap to filter and compare their performance.

## Building/Development
This is an app made with React and NEXT.JS.
1. Download and install [NodeJS](https://nodejs.org/en/download/current).
2. Clone the Property-Aggregator repository: `git clone https://github.com/jordanbrotherton/Property-Aggregator.git`
3. To launch for development: use `npm run dev` and navigate to the URL provided by the command.
4. To build: use `npm run build`.
